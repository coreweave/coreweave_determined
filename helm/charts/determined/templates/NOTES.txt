To submit experiments please install the Determined CLI locally: `pip install determined`.

The client is configured to communicate with the master via an environment variable:
export DET_MASTER={{ .Release.Name }}.{{ .Release.Namespace }}.{{ .Values.region | lower }}.ingress.coreweave.cloud:80

To access the WebUI, please visit: https://{{ .Release.Name }}.{{ .Release.Namespace }}.{{ .Values.region | lower }}.ingress.coreweave.cloud

Login information:

   Username: admin
   Password: <BLANK>

To change the password or create/add/edit users, you can do via the CLI:

Example (change password):

det user change-password admin

Example (create a user with admin priveleges):

det user create --admin newuser (will create newuser with Admin priveleges)

Example (login a user):

det user login
